# Yamanoha — The Mountain Silhouette Viewer

**山の端** (*Yamanoha*) is a Japanese word for the faint line where mountain ridges meet the sky —
a sight that has moved poets and wanderers for centuries.

This project reconstructs that silhouette computationally.
Standing at north town in Tokyo, and looking westward,
one can see the layered ridgelines of the Tanzawa, Okumusashi, Okutama, and Chichibu ranges,
with Mount Fuji appearing on clear days — yet few can name which peak is which.
